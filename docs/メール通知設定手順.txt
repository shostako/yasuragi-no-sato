========================================
メール通知設定手順（Resend）
========================================

このホームページでは、新規予約・お問い合わせがあった際に
メールで通知を受け取ることができます。

【使用サービス】
Resend（https://resend.com）
- 月3,000通まで無料
- 設定簡単
- 高い到達率

========================================
1. Resendアカウントを作成する
========================================

1. https://resend.com にアクセス
2. 「Get Started」をクリック
3. GitHubまたはメールでアカウント作成
4. メール認証を完了

========================================
2. APIキーを発行する
========================================

1. ダッシュボードにログイン
2. 左メニューの「API Keys」をクリック
3. 「Create API Key」をクリック
4. 名前: 「やすらぎの郷」など分かりやすい名前
5. Permission: 「Sending access」を選択
6. 「Add」をクリック
7. 表示されたAPIキー（re_から始まる文字列）をコピー

【重要】APIキーは一度しか表示されません。必ずコピーして保存してください。

========================================
3. Vercelに環境変数を設定する
========================================

1. https://vercel.com にアクセス
2. プロジェクト「yasuragi-no-sato」を選択
3. 上部メニューの「Settings」をクリック
4. 左メニューの「Environment Variables」をクリック
5. 以下を入力して「Add」：

   【1つ目】
   - Key: RESEND_API_KEY
   - Value: （コピーしたAPIキーを貼り付け）
   - Environment: Production, Preview, Development すべてにチェック

   【2つ目】
   - Key: ADMIN_EMAIL
   - Value: （通知を受け取りたいメールアドレス）
   - Environment: Production, Preview, Development すべてにチェック

6. 「Save」をクリック

========================================
4. 再デプロイする
========================================

環境変数を反映させるため、再デプロイが必要です。

1. プロジェクトの「Deployments」タブをクリック
2. 最新のデプロイの右側「...」メニューをクリック
3. 「Redeploy」をクリック
4. 確認ダイアログで「Redeploy」をクリック

========================================
5. 動作確認
========================================

1. ホームページの予約フォームまたはお問い合わせフォームから
   テスト送信を行う
2. 設定したメールアドレスに通知が届けば設定完了

※送信元は「onboarding@resend.dev」になります
※独自ドメインからの送信も可能（Resendでドメイン認証が必要）

========================================
トラブルシューティング
========================================

【通知が届かない場合】
- 迷惑メールフォルダを確認
- Vercelの環境変数が正しく設定されているか確認
- 再デプロイを行ったか確認
- Resendダッシュボードの「Logs」で送信履歴を確認

【APIキーを再発行したい場合】
- Resendダッシュボードで既存キーを削除
- 新しいAPIキーを発行
- Vercelの環境変数を更新
- 再デプロイ

========================================
通知の内容
========================================

【予約の場合】
件名: 【新規予約】〇〇様 - 2026-01-20 午前の部
本文:
- お名前
- メールアドレス
- 電話番号
- 希望日・時間帯
- 見学人数

【お問い合わせの場合】
件名: 【新規お問い合わせ】〇〇様
本文:
- お名前
- メールアドレス
- お問い合わせ種別
- お問い合わせ内容

詳細は管理画面（/admin/reservations, /admin/contacts）で確認できます。
